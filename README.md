# Student Grade Calculator

A Python application that manages student records, calculates percentages and grades, and generates performance reports.

## Features

- Add student records
- Calculate percentage automatically
- Generate grades based on marks
- Store and retrieve student data using CSV files
- Object-Oriented Programming implementation
- Input validation and error handling

## Technologies Used

- Python 3
- CSV File Handling
- Object-Oriented Programming (OOP)

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100   | A+    |
| 80 - 89    | A     |
| 70 - 79    | B     |
| 60 - 69    | C     |
| 50 - 59    | D     |
| Below 50   | F     |
